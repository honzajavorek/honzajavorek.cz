Webdesignérovy nepostradatelné odkazy

  • 26. 4. 2008
  • Honza Javorek

Upozornění! Tento článek vyšel před 14 lety. Můžete si jej přečíst v rámci zkoumání minulosti, ale přepokládejte, že dnes se Honza již nemusí s obsahem ztotožňovat.

Na internetu jsou seznamy se stovkami užitečných stránek a nástrojů pro tvůrce webů. No, popravdě mi nikdy k ničemu nebyly, protože než bych je proklikal nebo v nich něco našel… Navíc každý má „ten svůj“ seznam užitečných odkazů a jiným nemusí vyhovovat. Nejlepší je tedy nechat čas ukázat, které weby jsou nepostradatelné a užitečné. Já se o svůj seznam podělím a třeba se mi povede doplnit ten váš. A třeba se vám v komentářích povede doplnit ten můj ;) .

  • Google Analytics, Google Webmaster Tools, Feedburner – nástroje pro sledování úspěšnosti a návštěvnosti mých projektů
  • FavIcon from pics – dvě kliknutí a máte favicon
  • POW – seznam levných a bezplatných fotobank
  • ColourLovers – no comment :)
  • Full page test, Analýza zdrojového kódu, kodérův checklist (mám svůj upravený) – pro lepší a čistější kód ;)
  • MySQL v kostce, SQL Designer – pro databázisty
  • United colors… – barvičky a jejich význam (rychlý tahák)
  • Getting real – bible pro webové projekty
  • Houvnou – různé triky s PHP apod. od Davida Grudla
  • přehled standardů W3C – někdy je hezké mít jasno
  • seznam CSS galerií – pokud se chcete chlubit se svými webíšky
  • 960 grid system – je libo mřížka?
  • gotAPI – API úplně na cokoliv
